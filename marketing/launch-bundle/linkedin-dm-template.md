## Best Practices for LinkedIn DMs

### DO:
1. **Personalize the first line** — Reference their company, recent win, or specific vertical
2. **Keep it conversational** — LinkedIn DMs should feel like you're talking to a colleague, not sending a sales pitch
3. **Use bullet points** — Makes it scannable (people read DMs on mobile)
4. **Ask questions** — Questions get responses, statements get ignored
5. **Be specific** — "Your competitors won $8M in cyber work" beats "Competitors are winning"
6. **Respond fast** — If they reply, respond within 30 minutes if possible

### DON'T:
1. **Pitch immediately** — Build rapport first (connection note → intro message → value offer)
2. **Write essays** — Keep DMs under 150 words
3. **Use generic templates** — "Hi [First Name], I see you work in government contracting..." = instant ignore
4. **Be pushy** — If they don't respond after 3 touches, let it go
5. **Use emojis excessively** — 1-2 is fine, 10+ looks unprofessional
6. **Ignore their responses** — If they ask a question, answer it thoroughly

## Metrics to Track

| Metric | Target | How to Measure |
|--------|--------|----------------|
| **Connection Accept Rate** | 30-50% | LinkedIn "Pending" tab |
| **First Message Open Rate** | 60-80% | LinkedIn read receipts (if enabled) |
| **Reply Rate** | 20-30% | Count responses / sent messages |
| **Link Click Rate** | 15-25% | URL tracking (Bitly or similar) |
| **Free Signup Rate** | 10-15% of replies | Beehiiv dashboard |

**Weekly DM Volume Target:** 20-30 new connections + 10-15 follow-ups = 30-45 DMs/week

**Expected Conversion:**
- 30 connection requests → 12 accepts (40%) → 4 replies (33%) → 1-2 free signups (25-50% of replies)

**That's 3-6% connection-to-signup conversion, which is solid for cold LinkedIn outreach.**
